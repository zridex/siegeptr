Updater flow

How it works

The updater uses the same files as the manual downloads page. It just handles the folder replacement step for you.

  1. Close the PTR client.
  2. Open the updater and confirm the PTR AddOns folder.
  3. Click Check updates.
  4. Select the addon families you want.
  5. Click Update selected. Existing folders are backed up first.
Prefer manual installs? The zip downloads are still available on the Addon downloads page.
