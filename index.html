Install

  1. Close the PTR client.
  2. Unzip the download.
  3. Open your World of Warcraft install folder.
  4. Copy the addon folders into the PTR AddOns folder below.
  5. Enable out-of-date addons if the client asks.
Inside your WoW folder _classic_ptr_\Interface\AddOns
